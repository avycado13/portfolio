

# avycado13's Portfolio

This is the personal portfolio website of [avycado13](https://github.com/avycado13), showcasing projects, skills, and a bit of personality. Built with modern web tools and designed to be fast, responsive, and visually engaging.

## 🌟 Features

- Responsive project cards with images and modals
- Dark mode toggle with theme persistence
- Tag filtering for projects
- Loading skeleton UI for better UX
- Open Graph and Twitter meta tags for sharing
- GitHub link with dark/light icon switching
- Footer with attribution and social link

## 🛠 Tech Stack

- [Github Pages](https://pages.github.com/) – static site hosting
- [Tailwind CSS](https://tailwindcss.com/) – utility-first CSS
- [daisyUI](https://daisyui.com/) – Tailwind component library
- Vanilla JavaScript – lightweight and fast

## 📁 Structure

- `index.html` – Main page
- `projects.json` – Project data source (served or embedded)
- `github-mark.png` / `github-mark-white.png` – GitHub logo for light/dark modes
